#### HTML Text Formatting

```html
<b>	กำหนดข้อความแบบตัวหนา
<big>	กำหนดข้อความแบบตัวใหญ่
<del>	 กำหนดข้อความที่ถูกลบ
<em>	กำหนดข้อความแบบตัวเอียง
<i>	กำหนดข้อความแบบตัวเอียง
<ins>	กำหนดข้อความที่ถูกแทนที่
<small>	กำหนดข้อความแบบตัวเล็ก
<strong>	กำหนดข้อความแบบตัวหนา
<sub>	กำหนดข้อความแบบตัวห้อย
<sup>	กำหนดข้อความแบบตัวยก
```

```html
<html>
<body>

<p><b>ข้อความตัวหนา </b></p>
<p><big>ข้อความตัวใหญ่</big></p>
<p><i>ข้อความตัวเอียง </i></p>
<p><code>ข้อความตัวคอมพิวเตอร์</code></p>
<p><sub>ตัวห้อย</sub> และ<sup>ตัวยก</sup></p>
<p><b><i>ตัวหนาเอียง</i></b></p>
<p><small>ข้อความตัวเล็ก</small></p>
ภาษา <abbr title="Hyper Text Markup Language">HTML</abbr> ไว้สำหรับสร้างเว็บเพจ.

</body>
</html>
```
